P2P Games - Taş Kağıt Makas

1. Oyuna katılmak için HPOT tokenlarınızı yatırın

2. Rakibiniz ile eşleşmeyi bekleyin

3. Seçiminizi yapın: Taş, Kağıt veya Makas

4. Kazanan, toplam bahis miktarının %98'ini alır (%2 platform ücreti)

Against Case Games - Slot

1. Bahis miktarınızı belirleyin

2. Slot makinesini çalıştırın

3. Kazanç oranları panelden görüntülenebilir

4. Tüm sonuçlar blockchain üzerinde doğrulanabilir

Community Games - Jackpot

1. Havuza HPOT token yatırın

2. Her token size bir şans verir

3. Süre dolduğunda rastgele bir kazanan seçilir

4. Kazanan, havuzun %98'ini alır (%2 platform ücreti)

Election Game - Club Election

1. Adaylar için oy kullanın

2. Her oy için HPOT token yatırın

3. Kazanan adaya oy verenler ödül havuzunu paylaşır

4. Dağıtım, yatırılan token miktarına göre yapılır

Genel Bilgiler

• Tüm oyunlar akıllı kontratlar üzerinde çalışır

• Sonuçlar blockchain üzerinde şeffaf bir şekilde görüntülenebilir

• Platform ücreti tüm oyunlarda %2'dir

• Minimum bahis miktarı 10 HPOT'dur

• Kazançlarınızı anında çekebilirsiniz
